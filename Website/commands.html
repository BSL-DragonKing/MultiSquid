Help Page

This is the place where you can find help for all the commands you can use with MultiSquid


General

Command Description
Help Brings up an embed help message
Invite Give you the invite link to invite this client to a server of your choice
Support Gives you the link to offical support server for MultiSquid
About Brings out an embed message of a brief description about the bot
Website Brings up the offical MultiSquid Website

Moderation

Command Description
Clear Clears a certain amount of messages and provide a number after the argument (If a number isn't the client default is set to clear 5 messages)
Kick Kicks a member from your server (Use the @ to kick the member)
Ban Bans a member from your server (Use the @ to ban the member)
Unban Unbans user from your server (To use type "Discord Username of banned user + Number of their Discord Discriminator")
Mute Mutes a certain member (To use this please @ the person after the command)
Unmute Unmutes a certain member (To use this please @ the person after the command)

Reminder

Command Description
Reminder Brings up my reminders page for you
Date Brings up today's date
Time Brings up just the time
Mtime Similar to time but brings up the time in military form
